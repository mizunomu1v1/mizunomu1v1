### リポジトリ作成方法

1. GitHubで作成
2. VSCodeでCtrl + Shift + P
3. Git Clone
4. GithubからPATHをコピペ
5. ローカル作業フォルダに複製

## Spring Boot

###  導入

1. コマンドパレットから**Spring Initializr: Create a Maven Project**を選択
2. バージョンは最新を選択
3. Javaを選ぼう
4. Group IDはなんでもいいよ（パッケージ名）
5. Artifact IDもなんでもいいよ（プロジェクトを識別するID）
6. jarを選ぼう
7. 一番上（17）でいいぜ
8. 依存関係は用途に合わせるのだ！
